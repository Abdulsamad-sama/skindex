# Skindex — Precision Skin Science

![Skindex Logo](/public/skindex-1.png)

Skindex is an AI-driven skin analysis platform designed to provide precision metrics for your unique skin profile. Optimized for every skin tone, Skindex offers a comprehensive suite of tools to help you understand and care for your skin better.

## 🌟 Key Features

- **AI-Powered Skin Analysis**: Advanced detection and analysis of skin metrics (texture, hydration, tone, and more) using state-of-the-art computer vision.
- **Dynamic Health Dashboard**: A centralized view of your skin's health progress, metrics, and personalized insights.
- **Weather Routine Recommendations**: Smart, weather-aware skincare and outfit suggestions tailored to your local environmental conditions.
- **Precision for All Tones**: Engineered to be inclusive and accurate across the full spectrum of skin tones.
- **Outfit Recommendations**: Stylized suggestions that complement your skin routine and daily weather.

## 🚀 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm / pnpm / yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Abdulsamad-sama/skindex.git
   cd skindex-app
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🛠️ Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) & [Base UI](https://base-ui.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **AI/CV**: [MediaPipe](https://google.github.io/mediapipe/) for face detection and analysis.

## 📂 Project Structure

- `/app`: Next.js App Router pages and layouts.
- `/components`: Reusable UI components.
- `/public`: Static assets (images, icons).
- `/lib`: Utility functions and shared logic.
